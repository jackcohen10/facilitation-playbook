## How to Use

1. Get an OpenAI API key from [platform.openai.com](https://platform.openai.com/api-keys)
2. Enter your API key (it's saved in your browser)
3. Describe what you need in natural language
4. Click "Find Exercises" to get AI-powered recommendations
5. Use theme filters to narrow down results
6. Copy exercise instructions or print results

## Technology
